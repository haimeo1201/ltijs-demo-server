# Ltijs Demo Server

> Ltijs v5 demo server

### Usage

- Download or clone the repo

- Setup `.env` file with the relevant variables

  ```
  DB_HOST=
  DB_AUTHSOURCE=
  DB_NAME=
  DB_USER=
  DB_PASS=
  LTI_KEY=
  NODE_TLS_REJECT_UNAUTHORIZED=0
  PORT=8080
  CLIENT_ID=
  ISS=
  PLATFORM_URL=
  ```

- Run `npm install`

- Run `npm start`
